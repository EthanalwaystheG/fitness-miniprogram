# 设计规范

## 色彩系统

| 变量名 | 色值 | 用途 |
|--------|------|------|
| `--color-primary` | `#07C160` | 主色（微信绿），用于强调、按钮、BMR/TDEE 数值 |
| `--color-primary-light` | `#E6F9EF` | 主色浅底 |
| `--color-bg` | `#F5F5F5` | 页面背景 |
| `--color-card` | `#FFFFFF` | 卡片背景 |
| `--color-text` | `#333333` | 主文字 |
| `--color-text-secondary` | `#999999` | 次要文字 |
| `--color-text-light` | `#B0B0B0` | 辅助/占位文字 |
| `--color-danger` | `#FA5151` | 删除/警告/体重上升 |
| `--color-warning` | `#FFC300` | 警告色 |
| `--color-protein` | `#FF6B6B` | 蛋白质标识色 |
| `--color-carbs` | `#FFD93D` | 碳水标识色 |
| `--color-fat` | `#6BCB77` | 脂肪标识色 |

## 间距系统

| 变量名 | 值 | 用途 |
|--------|-----|------|
| `--spacing-xs` | `8rpx` | 最小间距 |
| `--spacing-sm` | `16rpx` | 卡片间隙、内边距 |
| `--spacing-md` | `24rpx` | 标准内边距、卡片 padding |
| `--spacing-lg` | `32rpx` | 大内边距、区块间距 |
| `--spacing-xl` | `48rpx` | 超大间距 |

## 圆角

| 变量名 | 值 | 用途 |
|--------|-----|------|
| `--radius` | `16rpx` | 卡片、按钮 |
| `--radius-lg` | `24rpx` | 大圆角元素 |

## 字体层级

| 用途 | 大小 | 字重 |
|------|------|------|
| 页面标题 | `34rpx` | 600 |
| 区块标题 | `32rpx` | 600 |
| 日期文字 | `30-32rpx` | 600 |
| 正文/列表项 | `28rpx` | 400-500 |
| 大数字 (BMR/TDEE) | `44-48rpx` | 700 |
| 超大数字 (热量总计) | `56rpx` | 700 |
| 辅助文字 | `24-26rpx` | 400 |
| 标签/说明 | `20-22rpx` | 400 |

## 组件设计规范

### 卡片 (.card)
- 白色背景 `#FFF`
- 圆角 `16rpx`
- 内边距 `24rpx`
- 左右外边距 `16rpx 24rpx`
- 阴影 `0 2rpx 12rpx rgba(0,0,0,0.04)`

### 按钮 (.btn-save / .btn-confirm)
- 主色背景 `#07C160`
- 白色文字
- 圆角 `12rpx`
- 内边距 `20rpx 0`

### FAB (浮动操作按钮)
- 圆形 100rpx × 100rpx
- 主色背景 + 白色 "+"
- 带阴影 `0 8rpx 24rpx rgba(7,193,96,0.35)`
- 固定在右下角

### 弹窗 (editor)
- 半透明遮罩 `rgba(0,0,0,0.45)`
- 底部弹出面板，圆角 `32rpx 32rpx 0 0`
- 最大高度 80-85vh
- 底部安全区适配 `env(safe-area-inset-bottom)`

## Tab Bar
- 未选中色 `#999999`
- 选中色 `#07C160`（微信绿）
- 背景白色
- 当前为纯文字标签模式（无图标）

## 页面布局
- 全页面滚动 `<scroll-view scroll-y>`
- 高度 `100vh`
- 底部留白 `40rpx` 避免内容被遮挡
- 自定义导航栏 + Tab Bar 自动适配
